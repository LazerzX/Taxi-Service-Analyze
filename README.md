# Taxi Service Analysis System

ระบบการวิเคราะห์การบริการแท็กซี่(Taxi Service Analysis System) 
เป็นระบบการวิเคราะห์การทำงาน การขับและการบำรุงเบื้องต้นของอู่แท็กซี่แห่งหนึ่ง มีวัตถุประสงค์เพื่อใช้ในการวิเคราะห์แผนการใช้งานทรัพยากรต่างๆ การซ่อมบำรุงตามปัจจัยต่างๆเบื้องต้น การวิเคราะห์ทางธุรกิจให้กับผู้เช่ารถเพื่อให้ผู้ขับและเจ้าของอู่รถสามารถสร้างรายได้และเดินรถได้อย่างมีประสิทธิภาพสูงสุด

# คุณลักษณะและขอบเขต

**คุณลักษณะขั้นต่ำ**
ทำให้โปรแกรมการวิเคราะห์ข้อมูลสามารถวิเคราะห์การเดินรถ การใช้ทรัพยากรต่างๆ พฤติกรรมของผู้ขับได้ สามารถเพิ่ม ลบและแก้ไขข้อมูลของรถทั้งพฤติกรรมผู้ขับและการเดินรถได้ การวิเคราะห์รายได้การเดินรถในเขตต่างๆในช่วงเวลาที่ต่างกันเพื่อนำมาใช้วิเคราะห์ข้อมูล ให้เกิดประโยชน์สูงสุด การสร้างและแสดง Dashboard หรือ Table เพื่อเก็บสถิติในระดับสัปดาห์ เดือนและปีเพื่อใช้วิเคราะห์ข้อมูลในระยะยาวได้
**คุณลักษณะเพิ่มเติม**
การจัดอันดับสถิติเขตที่ทำรายได้ในแต่ละช่วง การทำบันทึกประวัติการซ่อม การแสดงข้อมูลค่าใช้จ่ายการซ่อมบำรุงของรถแต่ละคัน

# ความท้าทายและความเสี่ยง

- ยังต้องใช้เวลาในการศึกษาในการทำ GUI | Solution: ศึกษาเนื้อหาเพิ่มเติมจากข้อมูลจากทั้งในเนื้อหาเรียน เนื้อหาจากทางเว็บไซต์และการถามผู้เชี่ยวชาญระดับสูง เช่น ระบบ AI
- การออกแบบระบบเพิ่มเติมให้สอดคล้อง | Solution: ออกแบบระบบพื้นฐานให้รองรับการขยายส่วนต่างๆล่วงหน้า
- การจัดการความผิดพลาดทำให้ต้องเสียเวลาเพิ่มเติมและต้องเรียนรู้และเฃื่อมส่วนประกอบใหม่ | Solution: วางแผนงานปัจจุบันและอนาคตพร้อมกันเพื่อรองรับการแก้ไขและเพิ่มเติมส่วนต่างๆ และติดตามความคืบหน้าของโครงงาน
